                 Understanding the Investigation Report
            

            

                

                    The JSON report represents structured digital forensics data collected and analyzed
                    during the investigation. Each field provides insight into evidence, malware behavior,
                    and investigative metrics.
                


                

                    

                        
 Case ID

                        

                            Unique identifier for tracking the investigation across systems and reports.
                        

                    


                    

                        
 Total Evidence

                        

                            Digital artifacts collected: logs, files, network traces, and system images.
                        

                    


                    

                        
 Analysis Results

                        

                            Completed forensic analyses including malware scans and behavioral analysis.
                        

                    


                    

                        
 Malware Samples

                        

                            Detected malicious software samples used to understand attack patterns.
                        

                    


                    

                        
 Evidence Types

                        

                            Categorization by type: network logs, system files, emails, memory dumps.
                        

                    


                    

                        
 Malicious Activities

                        

                            Detected threats: phishing, data breaches, malware attacks, identity theft.
                        

                    


                    

                        
 Investigation Metrics

                        

                            KPIs: duration, severity level, confidence score for effectiveness evaluation.
                        

                    


                    

                        
 Tools Used

                        

                            Forensic tools applied: malware scanners, IDS, analysis frameworks.
                        

                    

                


                

                     Note:
                    This structured JSON format ensures consistency, automation readiness,
                    and easy integration with AI-based threat analysis systems.
