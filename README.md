# Digital Forensics for Detecting and Investigating Cyber Malicious Activities

## 📌 Project Overview
This project is a Django-based Digital Forensics Investigation System designed to simulate the process of collecting, analyzing, and reporting digital evidence related to cyber malicious activities. The system supports evidence uploads, rule-based analysis, and structured forensic report generation with visualizations.

---

## 🚀 Features
- User and Admin Modules
- Evidence Upload and Management
- Rule-Based Threat Analysis
- Malware Classification
- Structured Forensic Reports
- Data Visualization and Charts
- CSV Report Export
- Multi-source Evidence Support
  - Computer
  - Mobile Devices
  - Network Logs
  - Cloud Services

---

## 🛠️ Technologies Used
- Python
- Django
- HTML
- CSS
- SQLite
- Matplotlib
- Visual Studio Code

## 📂 Supported Evidence Formats
